# Network Analyzer

Анализатор сетевых подключений — кроссплатформенное приложение для анализа сетевых интерфейсов и URL-адресов, разработанное на C# с использованием Avalonia UI.

## Описание

Приложение предоставляет удобный интерфейс для:
- Просмотра всех сетевых интерфейсов компьютера
- Детального анализа URL-адресов
- Проверки доступности хостов (Ping)
- Получения DNS-информации
- Сохранения истории проверок

## Функциональность

### Сетевые интерфейсы
- Отображение списка всех доступных сетевых интерфейсов
- Детальная информация по каждому интерфейсу:
  - Имя и описание интерфейса
  - IP-адрес
  - Маска подсети
  - MAC-адрес
  - Состояние подключения (Up/Down)
  - Скорость соединения
  - Тип интерфейса

### Анализ URL
- Парсинг URL с автоматическим добавлением схемы (http://)
- Отображение компонентов URL:
  - Схема (протокол)
  - Хост
  - Порт
  - Путь
  - Параметры запроса
  - Фрагмент
- Определение типа адреса (локальный/публичный/loopback)
- Получение DNS-информации
- Проверка доступности через Ping

### История
- Автоматическое сохранение всех проверенных URL
- Загрузка URL из истории для повторного анализа
- Отображение времени проверки

## Технологии

- .NET 10.0 — платформа разработки
- Avalonia UI 11.3.12 — кроссплатформенный фреймворк для создания интерфейсов
- CommunityToolkit.Mvvm 8.2.1 — реализация паттерна MVVM
- System.Net.NetworkInformation — работа с сетевыми интерфейсами и Ping

## Возможные проблемы и решения

### На macOS некоторые интерфейсы показывают статус Down

Это нормально для системных или туннельных интерфейсов. Реальные сетевые интерфейсы (Wi-Fi/Ethernet) обычно имеют имена `en0`, `en1` и статус **Up**.

### Ping может требовать специальных разрешений на macOS

При первом запуске может потребоваться разрешение на сетевую активность.
